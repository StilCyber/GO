Давай немного поговорим о типах данных
Сейчас тебе нужно знать о следующих трех:
1. int - это целые числа 
2. string - это строка
3. bool - это булевы значения, т.е. true или false (да или не да, Диего)

В языке есть операторы сравнения:
базовое сравнение - >, <, >=, <=
равенство - == 
неравенство = !=

Понятней станет на практическом примере
Кстати, // - это комментарий в коде, после двух косых черт пиши что угодно душе твоей 

*************************

package main 

import "fmt"

func main() { 
    fmt.Println(5 > 2) // true
    fmt.Println(5 < 8) // true
    fmt.Println(8 == 8) // true
    fmt.Println(8 != 8) // false
}

*************************
Кстати, отличай = от == 
Одно равно - это оператор присваивания, он позволяет установить значение переменной
Двойное равно - оператор сравнение, возвращает true если данные равны или false
