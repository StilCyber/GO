Одни из способов работы с ветвлением, когда есть много разных вариантов выбора, является оператор switch
Вспомни уже знакомую ситуацию с if else 

Здесь текст сообщения заранее вынесен в константы 
Оператор switch позволяет выбрать один из вариантов в зависимости от значения переменной 
В этом коде если пользователь введет 0, то выведется var1, и все остальные case будут проигнорированы 
После ключевого слова switch указывается переменная, затем в фигурных скобках
перечисляются возможные ее значения 
case 0 - означает случай, когда значение переменной - 0 

default означает, собственно, дефолтный вариант, когда ни в одном из case нет текущего значения переменной 
Если отработает один из case, то default запускаться не будет 

///////

package main

import "fmt"

const var1 = "Вы не промокнете на улице, но ходе поисков зонта вас огрели сковородой - вы получаете -20xp.\nВ добавок к этому, череда оскорблений снизила вашу силу воли на 20 единиц в течение 24 часов"
const var2 = "Вам удалось избежать побоев, но вы сильно промокли и к вечеру простудились.\nТеперь придется взять больничный и пару-тройку дней терпеть новые издевательства.\nВы получаете -60xp и -25% к психической сопротивляемости пока не поправитесь"
const var3 = "Определитель наконец с выбором - или 1, или 0"

func main() {
	var decision int

	fmt.Println("Ваш выбор:")
	fmt.Scan(&decision)

	switch decision {
	case 0:
		fmt.Println(var1)

	case 1:
		fmt.Println(var2)

	default:
		fmt.Println(var3)
	}
}
