Вывод следующий:
MakeData: 1
MakeData: 0
Print: 2

Что происходит 
По знакомой схеме (посмотрите defer_with_struct) подбираемся к вычислению аргументам функции Print
Здесь так хитро обставлено, что сначала выполняется функция, которая печатает 1 и возвращает структуру
И уже у структуру Data берется метод Print, куда в качестве аргумента заносится указатель 

Дальше создаем переменную со значением pointer = new(int)
Оно прокручивается в функции где печается собственно 0 

И уже затем метод Print берет указатель и достает по нему измененное value - 2
